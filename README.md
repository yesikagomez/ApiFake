# json-server-now

Getting json-server to work on now.sh in 2019

# Getting started.

```
git clone https://github.com/thedamian/json-server-now.git
cd json-server-now
```

Change the db.js file to meet your initial requirements

```
npm install
```

Run locally:

```
npm run start
```

Install now

```
npm install -g now
```

Deploy it:

```
now
```
